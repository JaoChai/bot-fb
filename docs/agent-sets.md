# Agent Sets

รายการ Agent Sets ทั้งหมด พร้อมตัวอย่างการใช้งาน

## 🎨 Frontend Development

### frontend-dev
- **Agent:** frontend-developer
- **Skills:** ui-ux-pro-max, react-query-expert
- **MCP:** Context7, Chrome

**ใช้เมื่อ:**
- สร้าง/แก้ไข React components
- ปรับ UI/UX design
- แก้ไข styling, layout
- Debug React Query issues

**ตัวอย่าง:**
```
"แก้ปุ่ม login ไม่ทำงาน"
"สร้าง dashboard page ใหม่"
"ปรับ responsive ให้รองรับ mobile"
```

## 🎯 Quick Decision Guide

| คำถาม | Set |
|-------|-----|
| UI/UX ไม่สวย? | frontend-dev |
| API ไม่ทำงาน? | backend-dev |
| Query ช้า? | database หรือ performance |
| Search ไม่เจอ? | rag-debug |
| Webhook ไม่ส่ง? | webhook-debug |
| Deploy ล้ม? | deployment |
| ก่อน commit? | code-review |
| Security concern? | security |

---

## 💡 Tips

1. **ใช้หลาย sets ร่วมกัน** - เช่น backend-dev → backend-test → code-review
2. **ดู auto-trigger rules** - บาง sets จะถูกเรียกอัตโนมัติ
3. **memory ก่อนเสมอ** - ก่อนทำงานใหม่ ค้นหางานเก่าก่อน
4. **ไม่แน่ใจ?** - ถามได้ Agent จะแนะนำ set ที่เหมาะสม
